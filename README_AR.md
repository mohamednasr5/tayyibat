## 📱 نظرة عامة

**طيبات** هو تطبيق ويب متقدم يجمع بين تكنولوجيا PWA (تطبيق الويب التقدمي) والذكاء الاصطناعي لتقديم تجربة فريدة في اكتشاف الوصفات والأطباق الشهية. يوفر التطبيق:

- 🍳 **وصفات متنوعة** من مختلف المطابخ العربية والعالمية
- 🤖 **نظام ذكي** للتوصيات المخصصة باستخدام تقنيات Machine Learning
- 📱 **تطبيق ويب متقدم** يعمل بدون اتصال بالإنترنت
- 🌙 **واجهة مستخدم عصرية** بتصميم جميل وسلس
- 🔍 **بحث ذكي** مع معالجة اللغة الطبيعية
- 💾 **حفظ سهل** للوصفات المفضلة
- ⭐ **تقييمات وتقييمات** من المستخدمين الآخرين

---

## 🚀 المميزات الرئيسية

### 1. **تصفح الوصفات**
- عرض وصفات متعددة مع صور وتفاصيل
- تصفية حسب الفئات (مقبلات، أطباق رئيسية، حلويات، إلخ)
- معلومات عن الوقت والصعوبة والتقييمات

### 2. **نظام البحث الذكي**
- بحث سريع ودقيق عن الوصفات
- معالجة اللغة الطبيعية (NLP)
- البحث الغامض (Fuzzy Search)
- اقتراحات ذكية أثناء الكتابة

### 3. **نظام التوصيات الذكي**
- توصيات مخصصة بناءً على:
  - سجل المشاهدة
  - الوصفات المحفوظة
  - التقييمات السابقة
  - تفضيلات المستخدم
- تصفية تعاونية (Collaborative Filtering)
- تصفية قائمة على المحتوى (Content-Based Filtering)

### 4. **تطبيق ويب متقدم (PWA)**
- تثبيت التطبيق على الجهاز
- العمل بدون اتصال بالإنترنت
- تجربة مشابهة للتطبيقات الأصلية
- وقت تحميل سريع جداً

### 5. **تحليل المشاعر**
- تحليل تعليقات المستخدمين
- فهم آراء المستخدمين حول الوصفات
- تحسين التوصيات بناءً على المشاعر

## 🛠️ الملفات وشرحها

### **index.html** - الملف الرئيسي
- **الحجم:** كامل التطبيق
- **الوصف:** يحتوي على:
  - HTML: الهيكل الأساسي للصفحة
  - CSS: جميع الأنماط والتصميمات
  - JavaScript: المنطق الأساسي والتفاعل

**الأقسام الرئيسية:**
1. **الرأس (Header):** العنوان والترحيب
2. **شريط البحث:** للبحث عن الوصفات
3. **مرشحات الفئات:** التصفية حسب نوع الطبق
4. **شبكة الوصفات:** عرض الوصفات بشكل شبكي
5. **نافذة التفاصيل:** عرض تفاصيل الوصفة
6. **التذييل:** معلومات التطبيق

---

### **sw.js** - Service Worker
- **الغرض:** تمكين العمل بدون اتصال بالإنترنت
- **الميزات:**
  - تخزين الصفحات والموارد
  - استجابة سريعة من الذاكرة المؤقتة
  - تحديث تلقائي للمحتوى الجديد
  - معالجة الأخطاء

**الأحداث المدارة:**
- `install`: تثبيت الخدمة وتخزين الموارد
- `activate`: تنظيف الذاكرة المؤقتة القديمة
- `fetch`: اعتراض طلبات الشبكة

---

### **manifest.json** - بيانات التطبيق
- **الغرض:** معلومات التطبيق وإعدادات التثبيت
- **يحتوي على:**
  - اسم التطبيق بالعربية والإنجليزية
  - وصف التطبيق
  - الألوان والرموز
  - إعدادات التثبيت
  - الاختصارات المخصصة
  - معالجة المشاركة

---

### **animations.css** - الحركات والانتقالات
- **الغرض:** جميع التأثيرات البصرية والحركات
- **يحتوي على:**
  - 30+ حركة مختلفة
  - انتقالات سلسة
  - تأثيرات التمرير
  - دعم الأجهزة القديمة

**الحركات الرئيسية:**
- `fadeIn / fadeOut`: التلاشي والظهور
- `slideUp / slideDown / slideLeft / slideRight`: الانزلاق
- `pulse`: النبض
- `bounce`: القفز
- `scale`: التكبير والتصغير
- `rotate`: الدوران
- والمزيد...

---

### **advanced-ai.js** - نظام الذكاء الاصطناعي
- **الغرض:** معالجة ذكية للبيانات والتوصيات
- **الفئات الرئيسية:**

#### **AdvancedAI (الفئة الرئيسية)**
```javascript
const tayyibatAI = new AdvancedAI();
```

**الميزات:**

1. **معالجة اللغة الطبيعية (NLP)**
   ```javascript
   tayyibatAI.processSearchQuery("حمص بطحينة لذيذ")
   ```
   - تقسيم النص (Tokenization)
   - استخراج الكيانات (Entity Extraction)
   - حساب الملاءمة (Relevance Scoring)

2. **نظام التوصيات**
   ```javascript
   const recommendations = tayyibatAI.generateRecommendations(recipes, 5);
   ```
   - يحسب درجة الملاءمة لكل وصفة
   - يرتبها حسب الأهمية

3. **البحث الغامض (Fuzzy Search)**
   ```javascript
   const results = tayyibatAI.fuzzySearch(query, recipes);
   ```
   - يجد الوصفات حتى مع الأخطاء الطباعية
   - يحسب التشابه بين النصوص

4. **تحليل المشاعر**
   ```javascript
   const sentiment = tayyibatAI.analyzeSentiment(comment);
   ```
   - يحدد ما إذا كان الرأي إيجابياً أم سلبياً
   - يحسب درجة الثقة

5. **تخزين البيانات المحلية**
   - حفظ تفضيلات المستخدم
   - تسجيل سجل البحث
   - حفظ التقييمات

## 🔧 كيفية الاستخدام

### **للمستخدمين العاديين:**

1. **افتح التطبيق**
   - اذهب إلى رابط التطبيق في المتصفح
   - أو ثبت التطبيق على جهازك

2. **ابحث عن وصفة**
   - اكتب اسم الطبق أو المكون
   - اضغط بحث
   - ستظهر النتائج الملاءمة

3. **اختر فئة**
   - استخدم أزرار الفئات لتصفية الوصفات
   - انقر على "الأطباق الرئيسية" مثلاً

4. **عرض التفاصيل**
   - انقر على "عرض الوصفة"
   - ستظهر المكونات وطريقة التحضير

5. **احفظ المفضلة**
   - انقر زر "حفظ"
   - ستظهر رسالة تأكيد

### **للمطورين:**

#### **تثبيت محلي:**
```bash
# نسخ الملفات
git clone https://github.com/yourusername/tayyibat.git

# الدخول للمجلد
cd tayyibat-main

# تشغيل خادم محلي (إذا لم يكن موجود)
python -m http.server 8000

# افتح المتصفح
http://localhost:8000
```

#### **التخصيص:**

**إضافة وصفة جديدة:**
```javascript
// في index.html - قسم recipes
const recipes = [
    {
        id: 9,
        title: 'اسم الوصفة',
        category: 'appetizers',
        description: 'الوصف',
        emoji: '🍳',
        time: '30 دقيقة',
        difficulty: 'سهل',
        rating: 4.5,
        reviews: 100,
        ingredients: ['مكون1', 'مكون2'],
        steps: ['خطوة1', 'خطوة2']
    }
];
```

**تخصيص الألوان:**

## 🐛 حل المشاكل الشائعة

### **المشكلة: التطبيق لا يعمل بدون إنترنت**
**الحل:** تأكد من:
1. تسجيل Service Worker بنجاح
2. أن ملف `sw.js` موجود
3. تفعيل HTTP عند الاختبار محلياً

### **المشكلة: الوصفات لا تظهر**
**الحل:**
1. افتح Console (F12)
2. تحقق من الأخطاء
3. تأكد من وجود بيانات الوصفات

### **المشكلة: البحث بطيء**
**الحل:**
1. قلل عدد الوصفات في الاختبار
2. استخدم فهرسة (Indexing)
3. قلل عدد المتغيرات المحسوبة
